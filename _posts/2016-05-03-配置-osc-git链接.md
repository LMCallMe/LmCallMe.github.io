## 配置git

``` bash
$git config --global user.name = "xxx"
$git config --global user.email "xx@xxx.xx"
```

## 生成 ssh-key

``` bash
$ssh-keygen -t rsa -C "xx@xxx.xx"
```

## 到git-osc 中设置公钥

略

## 检测设置结果

```bash
$ssh -T git@git.oschina.net
```

若返回如下信息则表明成功配置

```bash
Welcome to Git@OSC, yourname!
```

## 与osc联系

```bash
$git clone git地址
```
