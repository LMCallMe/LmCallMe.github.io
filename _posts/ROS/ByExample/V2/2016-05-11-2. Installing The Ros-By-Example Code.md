# 下载安装示例:

```bash
cd ~/catkin_ws/src
git clone https://github.com/pirobot/rbx2.git
cd rbx2
git checkout indigo-devel
cd ~/catkin_ws
catkin_make
source ~/catkin_ws/devel/setup.bash
```
# 更新示例

```bash
cd ~/catkin_ws/src/rbx2
git pull
cd ~/catkin_ws
catkin_make
source devel/setup.bash
```
